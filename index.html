Kefir.js

Kefir — is an FRP (functional reactive programming) library for JavaScript inspired by Bacon.js and RxJS with focus on high perfomance and low memory consumption.
It has nice Bacon.js like API and RxJS's performance / memory usage characteristics.

Kefir has GitHub repository, where you can send pull requests, report bugs, and have fun reading source code.

Download (0.1.12)

kefir.js ~ 30 kb
kefir.min.js ~ 5 kb (when gzipped)

You can also get edge version from GitHub, but use it at your own risk

NPM

npm install kefir

Bower

bower install kefir

As you can see docs aren't finished yet, but you can already get some info here. If you have questions feel free to talk to me on twitter, github issues or via email.

Create a property

emitterKefir.emitter()
Creates an emitter, that is ordinary property, but also has additional .emit(type, value) method. Then you can easily send events to property via emit(). 

var emitter = Kefir.emitter();
emitter.log(); // log events to console (see log)
emitter.emit('value', 1);
emitter.emit('error', 'e1');
emitter.emit('value', 2);
emitter.emit('error', 'e2');
emitter.emit('end');

=> [emitter] <value> 1
=> [emitter] <error> e1
=> [emitter] <value> 2
=> [emitter] <error> e2
=> [emitter] <end>

emitter is the easiest way to create general purpose property, but it doesn't give control over active state of property (see active state). If you want to create general purpose property and have control over active state, you should use fromBinder.

emptyKefir.empty()
Creates an empty property, that already ended and doesn't has any current value or error. 

property = Kefir.empty();
property.log();

=> [empty] <end:current>

constantKefir.constant(value)
Creates ended property, with specified current value. 

property = Kefir.constant(1);
property.log();

=> [constant] <value:current> 1
=> [constant] <end:current>

constantErrorKefir.constantError(error)
Creates ended property, with specified current error. 

property = Kefir.constantError('e1');
property.log();

=> [constantError] <error:current> e1
=> [constantError] <end:current>

laterKefir.later(wait, value)
Creates a property, that produces single value after wait milliseconds then ends. 

property = Kefir.later(1000, 1);
property.log();

=> [later] <value> 1
=> [later] <end>

intervalKefir.interval(interval, value)
Creates a property, that produces same value each interval milliseconds. Never ends. 

property = Kefir.interval(1000, 1);
property.log();

=> [interval] <value> 1
=> [interval] <value> 1
=> [interval] <value> 1
...

sequentiallyKefir.sequentially(interval, values)
Creates a property containing given values (array), delivered with given interval in milliseconds. Ends after all values delivered. 

property = Kefir.sequentially(1000, [1, 2, 3]);
property.log();

=> [sequentially] <value> 1
=> [sequentially] <value> 2
=> [sequentially] <value> 3
=> [sequentially] <end>
