Kefir.js

Kefir — is an FRP (functional reactive programming) library for JavaScript inspired by Bacon.js and RxJS with focus on high perfomance and low memory usage.
It has nice Bacon.js like API and RxJS's performance / memory usage characteristics.

Kefir has GitHub repository, where you can send pull requests, report bugs, and have fun reading source code.

Download (0.1.9)

kefir.js ~ 30 kb
kefir.min.js ~ 5 kb (when gzipped)

You can also get edge version from GitHub, but use it at your own risk

NPM

npm install kefir

Bower

bower install kefir

As you can see docs aren't finished yet, but you can already get some info here. If you have questions feel free to talk to me on twitter, github issues or via email.
